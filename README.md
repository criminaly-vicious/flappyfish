# Flapfish

## 1. Titulo do Jogo
Flapfish

## 2. Contexto
Flapfish e um jogo estilo endless runner em que voce controla um peixe que precisa atravessar obstaculos (canos) e coletar moedas para aumentar a pontuacao. O objetivo e sobreviver o maior tempo possivel, percorrendo a maior distancia.

## 3. Controles
- `Espaco`
- `Seta para cima (UP)`
- `Clique do mouse`

Qualquer um desses comandos faz o peixe "bater a nadadeira" para subir.
